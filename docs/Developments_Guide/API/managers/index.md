# Managers API

Managers act as the control layer of PyAutomation, coordinating interactions between data models, hardware interfaces, and application logic.

## Available Managers

- **[Alarm Manager](alarm_manager.md)**: Manages alarm lifecycle and events.
- **[Database Manager](db.md)**: Handles database connections and table registration.
- **[OPC UA Client Manager](opcua_client.md)**: Manages OPC UA client connections.
- **[State Machine Manager](state_machine.md)**: Registry for application state machines.
